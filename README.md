# E-commerce
E-commerce site using custom php MVC.

# Requirements

- PHP 7.^
- Composer 3.^

# Installation

```

composer install

```

## Setup Environment Variables

```

cp .env.example .env

```

## Production

```env

APP_DEBUG=Off

```
## Status

+ Finished
